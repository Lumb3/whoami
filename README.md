# WhoAmI Portfolio Template

A minimalist single-page portfolio template for a software or automation-focused personal brand.

## Run Locally

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Build

```bash
npm run build
```

## Customize

Most placeholder content lives in `src/data/portfolioData.js`.

- Update `[Your Name]`, role, tagline, links, and contact details.
- Add certificates by appending objects to `certificates`.
- Add projects by appending objects to `projects`.
- Add experience entries by appending objects to `experience`.

The UI is split into small reusable components in `src/components` and page sections in `src/sections`.
